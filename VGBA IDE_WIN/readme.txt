----------------
- Requirements -
----------------

- Debian based Linux distribution (Debian, Ubuntu, etc...), else: manually install visualboyadvance-gtk before or after install.sh

- super-user permissions

- Java JRE 1.7 (or higher) installed, not openJDK!!
	Verify Java version via command java -version
	If Java isn't installed or openJDK is installed,
	get Java JRE following this easy guide:
	http://tecadmin.net/install-oracle-java-8-jdk-8-ubuntu-via-ppa/


------------
- Binaries -
------------
MediaFire Folder: http://www.mediafire.com/?3t5vn2bg9kgqg


----------------
- Installation -
----------------

- Verify requirements

- Extract VBGA_IDE.tar.gz archive

- Excecute install.sh


----------
- Launch -
----------

- Launch app from "development" category in menu

- OR right click on VGBA_IDE_RUNABLE.jar (in /home/my_user_name/VGBA_IDE/) -> open with -> Oracle Java X Runtime

- If first launch causes file reading errors, please create manually "settings.cfg" file in home/my_user_name/ VGBA_IDE/ and write down the workspace path you want to work in (ex: /home/my_user_name/VGBA_WS). If file already exists, make sure the path it contains is correct.
